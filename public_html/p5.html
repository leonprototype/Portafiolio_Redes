Practicas de Laboratorio: Practica 5

"1. Objetivo General: Conocer, identificar y comprobar el funcionamiento de los cables de red estructurado."

Armado de cables de enthernet

Cables cruzados y directos armados

Conexion cables cruzados

Logica

Para probar el correcto funcionamiento de estos cables hay que tomar en cuenta que el directo nos sirve para conectar dispositivos diferentes, mientras que el cruzado permite establecer una conexión con dos dispositivos iguales. Para esta práctica se usará un cable cruzado para conectar dos laptops entre si y verificar que se haya logrado al tener las mismas direcciones IP. Los cables directos los probaremos con dos computadoras que se conectarán mediante un router y para verificarlo se usará el mismo método que el de los cables cruzados.
